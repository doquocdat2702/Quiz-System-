# Quiz System 🚀

A modern Quiz Management System built with React + Vite + TailwindCSS.

## ✨ Features

### Authentication
- Login
- Register
- Fake JWT Token
- Protected Routes
- Admin Routes

### Quiz System
- View Quiz List
- Start Quiz
- Submit Answers
- Auto Score Calculation
- Result Page
- History Result

### Admin Dashboard
- Dashboard Statistics
- Manage Quizzes
- Manage Questions
- Upload Question Images
- Delete Questions
- Fake CRUD with localStorage

### UI/UX
- Modern SaaS UI
- Responsive Design
- Glassmorphism Effects
- Smooth Hover Animation
- Gradient Background
- Sidebar Layout

---

# 🛠️ Technologies

- React
- Vite
- TailwindCSS
- React Router DOM
- React Icons
- localStorage
